Main Path
1. Пользователь запускает программу.
2. Программа отображает меню.
3. Пользователь выбирает пункт "Просмотр задач".
4. Программа загружает список задач.
5. Программа выводит номер задачи.
6. Программа поочередно выводит все параметры задачи.
7. Программа повторяет пункты 5-7, пока не закончит выводить все задачи.
8. Программа отображает пользовательское меню.

Alternative Path
3.1 Пользователь выбирает другой пункт.
3.1.1 Пользователь выходит из программы.
3.1.2 Программа сохраняет данные.

5.1 Журнал пуст.
5.2 Программа выводит сообщение, что журнал пуст.
5.3 Программа отображает пользовательское меню.
