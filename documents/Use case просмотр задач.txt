Main Path
1. Пользователь запускает программу.
2. Интерфейс отображает меню.
3. Пользователь выбирает пункт "Просмотр задач".
4. Загрузчик задач загружает список задач.
5. Интерфейс выводит номер задачи.
6. Интерфейс поочередно выводит все параметры задачи.
7. Интерфейс повторяет пункты 5-7, пока не закончит выводить все задачи.
8. Интерфейс отображает пользовательское меню.

Alternative Path
3.1 Пользователь выбирает другой пункт.
3.1.1 Пользователь выходит из программы.
3.1.2 Загрузчик задач сохраняет данные.

5.1 Журнал пуст.
5.2 Интерфейс выводит сообщение, что журнал пуст.
5.3 Интерфейс отображает пользовательское меню.
