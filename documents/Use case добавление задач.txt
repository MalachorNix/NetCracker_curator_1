Main Path
1. Пользователь запускает программу.
2. Интерфейс отображает меню.
3. Пользователь выбирает пункт "Добавление задачи".
4. Интерфейс запрашивает значение очередного входного параметра.
5. Пользователь вводит значение очередного входного параметра.
6. Интерфейс и создатель задач проверяют корректность введенного параметра.
7. Значение корректно.
8. Пункты 4-7 повторяются для всех параметров.
9. Создатель задач создает задачу по введенным параметрам.
10. Журнал добавляет задачу в список задач.

Alternative Path
3.1 Пользователь выбирает другой пункт.
3.1.1 Пользователь выходит из программы.
3.1.2 Загрузчик задач сохраняет данные.

7.1 Значение не корректно.
7.2 Интерфейс запрашивает параметр снова.
