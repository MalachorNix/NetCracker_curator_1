Main Path
1. Пользователь запускает программу.
2. Интерфейс показывает приветствие.
3. Интерфейс предлагает войти в программу, используя существующий логин и пароль пользователя, 
зарегистрироваться или выйти из программы.
4. Пользователь вводит логин.
5. Пользователь вводит пароль.
6. Валидатор проверяет введенные логин и пароль с базой данных пользователей.
7. Значения корректны.
8. Интерфейс показывает главное меню программы.

4.9 Пароли совпадают.

зарегистрироваться или выйти из программы.
