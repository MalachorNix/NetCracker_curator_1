Main Path
1. Пользователь запускает программу.
2. Программа отображает меню.
3. Пользователь выбирает пункт "Удаление задачи".
4. Программа запрашивает какую задачу пользователь хочет удалить.
5. Пользователь выбирает задачу.
6. Программа удаляет выбранную задачу.

Alternative Path
3.1 Пользователь выбирает другой пункт.
3.1.1 Пользователь выходит из программы.
3.1.2 Программа сохраняет данные.

5.1 Выбранной задачи не существует.
5.1.1 Программа оповещает пользователья об этом.
5.1.2 Программа предлагает выбрать другую задачу.

5.2 Пользователь не выбрал никакую задачу и вышел.
5.2.1 Программа отображает меню.
