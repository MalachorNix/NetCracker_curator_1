Main path
1. Пользователь запускает программу.
2. Программа отображает меню.
3. Пользователь выбирает пункт "Редактирование задачи".
4. Программа запрашивает какую задачу пользователь хочет отредактировать.
5. Пользователь выбирает задачу.
6. Программа отображает подменю для редактирвоания.
7. Пользователь выбирает какое поле будет редактировать.
8. Программа запрашивает значение данного поля.
9. Программа проверяет корректность введенного значения.
10. Значение корректно.
11. Программа создает копию выбранной задачи.
12. Программа изменяет у этой копии нужное поле на введеное.
13. Программа заменяет выбранную задачу на отредактированную.
14. Пункты 6-13 повторяются пока пользователь не выйдет из редактирвания.
15. Программа выходит из режима редактирования.

3.1.2 Программа сохраняет данные.

5.1.2 Программа предлагает выбрать другую задачу.

5.2 Пользователь не выбрал никакую задачу и вышел.
5.2.1 Программа отображает меню.

7.1 Выбранного поля не существует.
7.1.1 Программа оповещает пользователья об этом.
7.1.2 Программа предлагает выбрать другое поле для редактирования.

7.2 Пользователь не выбрал никакое поле и вышел.
7.2.1 Программа выходит из режима редактирования.

10.1 Значение не корректно.
10.2 Программа запрашивает значение снова.
