Main Path
1. Пользователь запустил программу.
2. Программа выводит название программы.
3. Программа предлагает на выбор одну из функций программы. Каждая функция имеет свой номер.
4. Программа повторяет пункт 3, пока не предложит все функции программы.
5. Программа дает пользователю выбрать функцию.
6. Пользователь выбирает функцию программы, введя ее номер.
7. Программа проверяет номер введенной функции.
8. Номер функции корректен. Такая функция существует.
9. Программа выполняет выбранную пользователем функцию.
10. Программа, после выполнения функции, выбранной пользователем, заново выполняет пункты 3-5 (т.е заново 
отображает меню), пока пользователь не закроет программу.

Alternative Path
8.1 Значение некорректно. Функции с таким номером нет.
8.2 Программа предлагает заново ввести номер функции.
