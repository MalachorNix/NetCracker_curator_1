Main Path
1. Пользователь запустил программу.
2. Интерфейс выводит название программы.
3. Интерфейс предлагает на выбор одну из функций программы. Каждая функция имеет свой номер.
4. Интерфейс повторяет пункт 3, пока не предложит все функции программы.
5. Интерфейс дает пользователю выбрать функцию.
6. Пользователь выбирает функцию программы, введя ее номер.
7. Интерфейс проверяет номер введенной функции.
8. Номер функции корректен. Такая функция существует.
9. Журнал выполняет выбранную пользователем функцию.
10. Интерфейс, после выполнения функции, выбранной пользователем, заново выполняет пункты 3-5 (т.е заново 
отображает меню), пока пользователь не закроет программу.

Alternative Path
8.1 Значение некорректно. Функции с таким номером нет.
8.2 Интерфейс предлагает заново ввести номер функции.
