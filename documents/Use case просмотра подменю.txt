Main Path
1. Пользователь выбирает в главном меню функцию программы, которая содержит подменю.
2. Программа предлагает на выбор одну из подфункций функции. Каждая подфункция имеет свой номер.
3. Программа повторяет пункт 2, пока не предложит все подфункции функции.
4. Программа дает пользователю выбрать функцию.
5. Пользователь выбирает подфункцию функции, введя ее номер.
6. Программа проверяет номер введенной подфункции.
7. Номер подфункции корректен. Такая подфункция существует.
8. Программа выполняет выбранную пользователем подфункцию.
9. Программа, после выполнения подфункции, выбранной пользователем, заново выполняет пункты 2-4 (т.е заново 
отображает подменю), пока пользователь не выйдет из подменю. 

Alternative Path
6.1 Значение некорректно. Подфункции с таким номером нет.
6.2 Программа предлагает заново ввести номер подфункции.
